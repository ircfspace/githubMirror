# GitHub Mirror

A Telegram bot written in Python for monitoring GitHub releases and sending the files to a channel.

## Features

- 🚀 Monitors GitHub repositories for new releases
- 📦 Downloads and uploads release files to Telegram channel
- 🔒 Calculates and displays SHA256 hashes for files
- 🔗 Supports inline keyboard buttons for easy navigation
- 📱 Supports multiple app store links (Google Play, App Store, Microsoft Store)
- ⚡ NO 50MB FILE SIZE LIMIT (uses Telethon/MTProto)
- 🎯 Markdown support for formatted messages
- 🔄 Automatic duplicate prevention
